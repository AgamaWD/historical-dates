








## Инструкция по запуску проекта
* установите [NodeJS](https://nodejs.org/en/) (версия 18+ (если не установлен)

* Проверьте: node --version и npm --version

* Откройте терминал в корне проекта
    cd historical-dates

* Установите зависимости
    npm install

* Запустите проект

    npm run preview (режим просмотра) либо npm run dev (режим разработки)

Браузер автоматически откроется на [http://localhost:8080] (http://localhost:8080)
